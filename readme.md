## Run Game

Clone the project

```bash
  git clone https://github.com/Bishnu-coder/Breakout-Game
```

Go to the project directory

```bash
  cd Breakout-Game
```

Install dependencies

```bash
  pip3 install -r requirements.txt
```

Start the Game

```bash
  python3 main.py
```
